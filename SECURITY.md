# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| Latest  | ✅        |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Email us at **security@trusta.dev** with:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

We aim to respond within 48 hours and will keep you updated as we work on a fix.

Once resolved, we'll credit you in the release notes unless you prefer to remain anonymous.
